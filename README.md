
# vim-coloredit

This plugin provides to edit `#rrggbb`, `rgb(rr,gg,bb)` and `rgba(rr,gg,bb,aa)`.
When you edit `rgba(rr,gg,bb,aa)`, This plugin can not support to edit the alpha value.
